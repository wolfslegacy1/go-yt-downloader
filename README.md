# Go YouTube Downloader

Um utilitário simples em Go para baixar vídeos do YouTube diretamente pelo terminal.

## Pré-requisitos

* Go 1.16 ou superior instalado.

## Instalação

1. Clone o repositório:
git clone https://github.com/SEU_USUARIO/go-yt-downloader.git
cd go-yt-downloader

2. Instale as dependências:
go mod tidy

## Como usar

1. Execute o programa:
go run main.go

2. O terminal solicitará a URL do vídeo. Insira o link e pressione Enter.

Os vídeos baixados serão salvos automaticamente na pasta downloads criada na raiz do projeto.
